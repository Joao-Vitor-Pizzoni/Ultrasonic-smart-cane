# Ultrasonic Smart Cane

This project is an assistive technology solution developed using Arduino and an HC-SR04 ultrasonic sensor to detect obstacles in front of the user.

The system measures the distance between the user and nearby obstacles. When the detected distance is less than 1 meter, a buzzer is activated to provide an audible warning.

The goal of this project is to assist visually impaired individuals by improving safe navigation and obstacle awareness using simple electronic components.

This project demonstrates the practical use of ultrasonic sensing, distance measurement, and real-time alerts using Arduino.

## Components

* Arduino
* HC-SR04 Ultrasonic Sensor
* Buzzer
* Connecting wires

## How it Works

The ultrasonic sensor continuously measures the distance to objects ahead. If an obstacle is detected within 1 meter, the buzzer emits a sound to alert the user.

This simple assistive system can help visually impaired individuals detect obstacles and move more safely in their environment.
